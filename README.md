# CherryBomb
学完HTML/CSS/JavaScript做了个练手小网页，并未完全完善

运行Cherry Bomb.html即可查看网页

index.css 和 index.js 分别是渲染的css和js样式代码

## 功能介绍
左边为导航栏，右边为内容区

目前主要完成的三个功能如下

2、完成了iframe渲染，实现了内容区查看PDF和网页

3、TO DO LIST 待办事项，运用localstorage本地存储事项
